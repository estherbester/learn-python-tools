How to get into IRC

Create your account

  1. Go to http://webchat.freenode.net
  2. For "Nickname", pick an available username (use only letters, numbers, hyphen or underscore)
  3. You can ignore the other fields
  4. Except, fill out the captcha, and hit "Connect"


Register your Nickname

  1. Pick an insecure password - WARNING:The password is transmitted in plain text which means literally anyone on the internet who wants to can read it, so please don't re-use a password you already use!
  2. At the bottom of the messaging window (it should look like the screenshot below), type the following:
    /msg nickserv register YOUR_VERY_INSECURE_PASSWORD YOUR_EMAIL_ADDRESS
  3. Check your email and verify that you are now registered by following their instructions (it'll tell you to type something like the following in the message window):
    /msg NickServ VERIFY REGISTER YOUR_NICKNAME VERIFICATION_CODE_FROM_EMAIL )
  4. all IRC commands begin with the forward slash. At the bottom of this page you'll find a few more basic commands that are handy to know.


Install XChat

  1. Download from http://xchat.org
  2. Install and open. You will get the following window. Fill in Nickname with the nickname you just registered. Fill in the other fields if you'd like.
  3. In "Networks", find and select "FreeNode"


  4. (Optional) Configure XChat so that you can Identify yourself to IRC and automatically join #cars-python whenever you open XChat.
  5. Otherwise, you have to do run these commands yourself every time:
  6. /msg nickserv identify YOUR_VERY_INSECURE_PASSWORD
  7. /join #cars-python

Helpful commands

To join a channel, type this IRC command in the message window (note that channels begin with the # sign):
/join #CHANNEL_NAME

Other useful commands
  • /help -- get help from the IRC bot
  • /join #CHANNEL_NAME -- join a channel
  • /me ACTION -- describe yourself performing an action
  • /msg OTHER_PERSON'S_NICKNAME YOUR_MESSAGE -- send someone a private message
  • /part -- leave a channel
  • /quit -- leave IRC
