Install Python 2.7 on Windows 7

  1. Download the installer at http://python.org (direct link to .msi file)
  2. Run the Python installer
  3. Follow the instructions on the screen (you can accept all the defaults and click "Next")
  4. Test that you have Python by opening "Python (command line)"
  5. You should see a terminal window with Python 2.7.4 at the top and a cursor that looks like this:>>> _
  6. To open the Python interpreter the next time you log into Windows, select "Python (command line)" from the Start menu (ignore the GUI one for now)

Install Gitbash

This will install git, a source control tool, and a Linux-like shell on your system. We will discuss what git is used for in class, but for now let's just get it installed.

  1. Download the installer linked here from Google code
  2. Run the installer: accepting the default location.
  3. - At "Adjusting your PATH environment", select the first option, "Use Git Bash only."
  4. - At "Configuring the line ending conversions," select the first option, "Checkout Windows-style, commit Unix-style line endings."
  5. After it's installed, you can run Git Bash from the Start Menu or using the Desktop icon that will have been created during installation

And you're done!

That's all for now! Don't worry if you don't know what to type in the command prompts or if any of this is unfamiliar. If you come to the first class with both these programs installed, we'll be able to start learning Python right away!

(Advanced) Add Python paths to your environment

This part is optional for now

  1. open Control Panel
  2. select System and Security
  3. select System
  4. On the left, find the link "Advanced System Settings"
  5. Click the button marked "Environment Variables..."
  6. A dialog will appear with two windows. In the bottom window, find the line labeled "PATH"
  7. Hit the "Edit" button.
  8. A smaller dialog will appear with a bunch of text in an editable line. At the END of the line, copy/paste this text (semicolon included):
    ;c:\python27\;c:\python27\scripts
    and save it.

  9. Now hit the "New.." button. There are two lines to populate -- in the top line, type:
    PYTHONPATH
    In the bottom line, type:
    C:\Python27\Lib;C:\Python27\DLLs;C:\Python27\Lib\lib-tk;

  10. Now you can use the Windows shell to run the Python interpreter:Press [Windows key] + r and enter "command" -- a command shell will open up. At the prompt, ty'e python and you should see the Python prompt as described above.
